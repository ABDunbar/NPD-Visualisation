# Norwegian Petroleum Directorate (NPD)
The [NPD](https://npd.no/en) provides publically accessible data for export.  The data is organised into tabs;
* Wellbore
* Licence
* BAA
* Field
* Discovery
* Company
* Survey
* Facility
* TUF
* Stratigraphy

This repository provides a visual summary of the table data.

# Exploration Summary
